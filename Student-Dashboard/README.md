# Student Management Dashboard


## Project Overview

Student Management Dashboard is a React application used to manage student records.


## Technologies Used

- React
- React Router DOM
- React Hook Form
- Zod Validation
- Axios
- Context API
- CSS


## Features


### Authentication

- Basic Login
- Protected Dashboard Route


### Student Management

- Add Student
- Edit Student
- Delete Student
- Search Student
- Filter by City
- Filter by Status
- Sort by Name
- Pagination


### Performance

- Lazy Loading
- Code Splitting
- Suspense Loader


### Forms

- React Hook Form
- Zod Validation


### API

- Axios Integration
- Loading State
- Error Handling
